# 📖 Wholework User Guide

Welcome to the Wholework User Guide. This guide is for developers evaluating or actively using Wholework with Claude Code and GitHub.

## Who This Guide Is For

- Individual developers who want to automate their GitHub workflow with Claude Code
- Teams adopting Wholework incrementally, one phase at a time
- Anyone who has installed Wholework and wants to run their first `/auto` command

If you are looking for internal architecture and contribution guidelines, see [CONTRIBUTING.md](../../CONTRIBUTING.md) and [docs/structure.md](../structure.md) instead.

| Page | What you will learn |
|------|---------------------|
| [Quick Start](quick-start.md) | Install Wholework and run your first `/auto` in 10–15 minutes |
| [Workflow Guide](workflow.md) | What each command does and when to use it |
